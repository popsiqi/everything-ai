## 🎨 Выбор темы

| Тема | Случай использования | Примеры замены |
|------|---------------------|----------------|
| **gpt** | Стиль Кремниевой долины, международные продукты | `Thinking...` `Generating tokens...` |
| **cn-ai** | Внутренние продукты, самоконтролируемый | `大模型推理中...` `国产大模型思考中...` |
| **enterprise** | Традиционная корпоративная трансформация | `智能分析中...` `数字化转型引擎运行中...` |

```javascript
// Стиль Кремниевой долины
init({ theme: 'gpt' })

// Стиль внутренней большой модели
init({ theme: 'cn-ai' })

// Стиль корпоративной трансформации (максимум юмора)
init({ theme: 'enterprise' })
```

## 🔧 Эффекты плагинов

### cursor - Мерцающий курсор

Добавляет мерцающий курсор после "Мышление...", как AI генерирует в реальном времени:

```
Мышление...█ → Мышление... → Мышление...█
```

### icon - AI-иконка

Заменяет spinner на иконку мозга 🧠, мгновенно AI-продукт:

```
⏳ → 🧠 (вращается)
```

### prompts - Промпты мышления

Автоматически показывает эти промпты (меняются каждые 2 секунды):

```
Вызов модели 405B параметров...
Глубокое мышление нейронной сети...
Модель триллион-параметров работает...
Вычисление механизма внимания Transformer...
Attention is All You Need...
```

**Включите все эффекты одной командой:**

```javascript
init({
  plugins: ['cursor', 'icon', 'prompts'],
  theme: 'enterprise'
})
```
